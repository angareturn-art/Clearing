# 배치 파일 사용 설명서

## 📋 제공 파일

| 파일명 | 용도 | 포트 |
|--------|------|------|
| `run.bat` | 전체 실행 (백엔드+프론트엔드) | 5000 + 5173 |
| `run-backend.bat` | 백엔드 서버만 실행 | 5000 |
| `run-frontend.bat` | 프론트엔드만 실행 | 5173 |

---

## 🚀 실행 방법

### **전체 실행 (권장)**
```
더블클릭: run.bat
```
- 백엔드 + 프론트엔드가 자동으로 2개 창에서 실행됨
- 자동으로 npm install도 수행

### **백엔드만 실행**
```
더블클릭: run-backend.bat
```
- 포트 5000에서 API 서버 실행
- 데이터베이스: `construction.db`

### **프론트엔드만 실행**
```
더블클릭: run-frontend.bat
```
- 포트 5173에서 개발 서버 실행
- API 프록시: `http://localhost:5000`

---

## 🔧 접근 주소

| 항목 | 주소 |
|------|------|
| **프론트엔드** | `http://localhost:3000` 또는 `http://localhost:5173` |

## ⚙️ 특징

✅ **한글 완벽 지원**
- UTF-8 인코딩 (한글 깨짐 없음)
- 모든 메시지가 한글로 표시

✅ **자동 설치**
- npm 패키지 자동 설치
- 첫 실행 시 필요한 모든 의존성 다운로드

✅ **오류 처리**
- 설정 오류 시 명확한 오류 메시지 표시
- 실패 시 로그 보존

---

## 📝 주의 사항

1. **Node.js 필수**
   - Node.js 14 이상 설치 필요
   - `node --version`으로 설치 확인

2. **포트 충돌**
   - 포트 5000, 5173, 3000이 사용 가능해야 함
   - 다른 프로그램이 점유 중이면 오류 발생

3. **폴더 구조**
   - 반드시 `Cleaning_V2` 폴더에서 실행
   - `server/`, `src/` 폴더가 있어야 함

## 🐛 문제 해결

### **"node 명령을 찾을 수 없습니다"**
→ Node.js 설치 필요: https://nodejs.org/

### **"포트가 이미 사용 중입니다"**
→ 기존 프로세스 종료 후 재시도

### **한글이 여전히 깨집니다**
→ 배치 파일 더블클릭 대신 우클릭 → "편집" → 저장 후 재실행

## 💡 팁

- `run.bat` 사용 시 백엔드 시작 후 1초 대기 후 프론트엔드 시작
- 각 터미널 창을 닫으면 해당 서버 종료
- 모든 창을 닫으면 완전 종료
