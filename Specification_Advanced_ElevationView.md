# 고도화 배치도(Advanced Elevation View) 상세 설계 명세

이 문서는 기존 배치도와 별개로 운영되는 '시각적 공정 현황판'의 표준을 정의합니다.

## 1. 디자인 철학 (Design Philosophy)
*   **Glanceability (한눈에 보기)**: 클릭 없이도 전체 현황을 파악할 수 있어야 함.
*   **Time-Centric (시점 중심)**: "언제 했는가"를 색상과 텍스트로 즉시 노출.
*   **Simple (단순함)**: 갱폼 등 부가 정보를 제외하고 박리제/청소에만 집중.

## 2. 시각적 표준 (Visual Standards)
### **세대(Cell) 표현**
| 상태 | 시각적 표현 | 비고 |
| :--- | :--- | :--- |
| **박리제 완료** | 짙은 블루 배경 + 중앙 화이트 날짜 | 예: `4/24` |
| **청소 진행 중** | 세대 내부에 하늘색 수평 프로그레스 바 | 진행률 % 함께 표시 |
| **최종 완료** | 초록색(Success) 배경 + 체크(v) 아이콘 | 박리/청소 모두 완료 시 |
| **최근 작업지** | 금일 작업된 세대는 골드색 테두리(`ring-2 ring-yellow-400`) | 활력 있는 화면 구성 |

### **동별 요약 바 (Building Summary)**
- 각 그리드 상단에 위치.
- `[박리제 85% | 청소 60%]` 형태의 통합 게이지 바 제공.
- 해당 동의 최종 업데이트 시각(예: `2시간 전`) 표시.

## 3. 동작 및 데이터 로직
*   **데이터 필터링**: 기존과 동일하게 `site_id` 기반 현장 격리.
*   **날짜 추출**: 각 세대의 작업 기록 중 가장 최근 날짜(`MAX(date)`)를 가져와 표시.
*   **상태 판별**: 
    - `oiling_records` 존재 시 -> 박리제 완료 상태.
    - `cleaning_records` 존재 시 -> 진행률에 따른 게이지 표시.
